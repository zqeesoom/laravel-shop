## Installing 请安命令顺序安装

```shell
//安装wechat
composer require "overtrue/laravel-wechat:~5.0"
//安装组件命令模式扩展
composer require shineyork/laravel-extend-database

$ composer require zqeesoom/laravel-shop -vvv
```
